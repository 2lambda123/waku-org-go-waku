
NAME:
   gowaku - A new cli application

USAGE:
   gowaku [global options] command [command options] [arguments...]

VERSION:
   0.7.0-08cabab

COMMANDS:
   generate-key              Generate private key file at path specified in --key-file with the password defined by --key-password
   generate-rln-credentials  Generate credentials for usage with RLN
   help, h                   Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --config-file value                                                  loads configuration from a TOML file (cmd-line parameters take precedence)
   --tcp-port value, --port value, -p value                             Libp2p TCP listening port (0 for random) (default: 60000) [$WAKUNODE2_TCP_PORT]
   --address value, --host value, --listen-address value                Listening address (default: "0.0.0.0") [$WAKUNODE2_ADDRESS]
   --max-connections value                                              Maximum allowed number of libp2p connections. (default: 50) [$WAKUNODE2_MAX_CONNECTIONS]
   --websocket-support, --ws                                            Enable websockets support (default: false) [$WAKUNODE2_WEBSOCKET_SUPPORT]
   --websocket-port value, --ws-port value                              Libp2p TCP listening port for websocket connection (0 for random) (default: 60001) [$WAKUNODE2_WEBSOCKET_PORT]
   --websocket-secure-port value, --wss-port value                      Libp2p TCP listening port for secure websocket connection (0 for random, binding to 443 requires root access) (default: 6443) [$WAKUNODE2_WEBSOCKET_SECURE_PORT]
   --websocket-address value, --ws-address value                        Listening address for websocket connections (default: "0.0.0.0") [$WAKUNODE2_WEBSOCKET_ADDRESS]
   --websocket-secure-support, --wss                                    Enable secure websockets support (default: false) [$WAKUNODE2_WEBSOCKET_SECURE_SUPPORT]
   --websocket-secure-key-path value, --wss-key value                   Secure websocket key path (default: "/path/to/key.txt") [$WAKUNODE2_WEBSOCKET_SECURE_KEY_PATH]
   --websocket-secure-cert-path value, --wss-cert value                 Secure websocket certificate path (default: "/path/to/cert.txt") [$WAKUNODE2_WEBSOCKET_SECURE_CERT_PATH]
   --dns4-domain-name value                                             The domain name resolving to the node's public IPv4 address [$WAKUNODE2_WEBSOCKET_DNS4_DOMAIN_NAME]
   --nodekey value                                                      P2P node private key as hex. [$WAKUNODE2_NODEKEY, $GOWAKU-NODEKEY]
   --key-file value                                                     Path to a file containing the private key for the P2P node (default: "./nodekey") [$WAKUNODE2_KEY_FILE]
   --key-password value                                                 Password used for the private key file (default: "secret") [$WAKUNODE2_KEY_PASSWORD]
   --staticnode value                                                   Multiaddr of peer to directly connect with. Option may be repeated [$WAKUNODE2_STATICNODE]
   --keep-alive value                                                   Interval of time for pinging peers to keep the connection alive. (default: 5m0s) [$WAKUNODE2_KEEP_ALIVE]
   --persist-peers                                                      Enable peer persistence (default: false) [$WAKUNODE2_PERSIST_PEERS]
   --nat value                                                          TODO: Not implemented yet. Specify method to use for determining public address: any, none ('any' will attempt upnp/pmp) (default: "any") [$WAKUNODE2_NAT]

   --circuit-relay                                                      Enable circuit relay service (default: true) [$WAKUNODE2_CIRCUIT_RELAY]
   --resource-scaling-memory-percentage value                           Determines the percentage of total accessible memory that wil be dedicated to go-waku. A dedicated node with a lot of RAM could allocate 25% or more memory to go-waku (default: 25) [$WAKUNODE2_RESOURCE_MEMORY_PERCENTAGE]
   --resource-scaling-fd-percentage value                               Determines the percentage of total file descriptors that wil be dedicated to go-waku. (default: 50) [$WAKUNODE2_RESOURCE_FD_PERCENTAGE]
   --log-level value, -l value                                          Define the logging level (allowed values: DEBUG, INFO, WARN, ERROR, DPANIC, PANIC, FATAL) (default: INFO) [$WAKUNODE2_LOG_LEVEL]
   --log-encoding value                                                 Define the encoding used for the logs (allowed values: console, nocolor, json) (default: console) [$WAKUNODE2_LOG_ENCODING]
   --log-output value                                                   specifies where logging output should be written  (stdout, file, file:./filename.log) (default: "stdout") [$WAKUNODE2_LOG_OUTPUT]
   --agent-string value                                                 client id to advertise (default: "go-waku") [$WAKUNODE2_AGENT_STRING]
   --relay                                                              Enable relay protocol (default: true) [$WAKUNODE2_RELAY]
   --topic value [ --topic value ]                                      Default topic to subscribe to. Argument may be repeated. Deprecated! Please use pubsub-topic and/or content-topic instead. [$WAKUNODE2_TOPICS]

   --storenode value                                                    Multiaddr of a peer that supports store protocol. Option may be repeated [$WAKUNODE2_STORENODE]
   --store                                                              Enable store protocol to persist messages (default: false) [$WAKUNODE2_STORE]

   --discv5-enr-auto-update                                             Discovery can automatically update its ENR with the IP address as seen by other nodes it communicates with. (default: false) [$WAKUNODE2_DISCV5_ENR_AUTO_UPDATE]

   --rendezvous                                                         Enable rendezvous protocol for peer discovery (default: false) [$WAKUNODE2_RENDEZVOUS]
   --rendezvous-node value                                              Multiaddr of a waku2 rendezvous node. Option may be repeated [$WAKUNODE2_RENDEZVOUSNODE]

   --rpc                                                                Enable the rpc server (default: false) [$WAKUNODE2_RPC]

   --rpc-relay-cache-capacity value                                     Capacity of the Relay REST API message cache (default: 30) [$WAKUNODE2_RPC_RELAY_CACHE_CAPACITY]
   --rpc-admin                                                          Enable access to JSON-RPC Admin API (default: false) [$WAKUNODE2_RPC_ADMIN]
   --rest                                                               Enable Waku REST HTTP server (default: false) [$WAKUNODE2_REST]
